# 🔴 Terraforming Mars Card Database

A complete, searchable card reference for **Terraforming Mars**, covering all Projects, Corporations, and Preludes across every expansion.

## 🌐 [View the Live Database](https://hadronikle.github.io/Complete-Terraforming-Mars-Card-Database/)

---

## Features

- **489 cards** across all expansions
- Browse **Projects**, **Corporations**, and **Preludes**
- Filter by expansion, toggle individual sets or view all at once
- Search cards by name, number, or expansion
- Click any card for a full-size lightbox view with keyboard navigation
- Fast and mobile-friendly

## Expansions Covered

| Expansion | Contents |
|-----------|----------|
| Base Set | Core project and corporation cards |
| Corporate Era | Additional projects and corporations |
| Venus Next | Venus-themed cards |
| Prelude | Prelude cards (sets 1 & 2) |
| Colonies | Colony-focused cards |
| Turmoil | Political and turmoil cards |
| Promo | Promotional cards |

## 📌 About the Scans

All cards have been physically scanned at 800DPI resolution and are suitable for printing out if you are missing a card in your collection. If you notice a missing card, a duplicate, or a quality issue, feel free to [open an Issue](https://github.com/hadronikle/Complete-Terraforming-Mars-Card-Database/issues) and let me know.
